Methodology & Research

Technical specifications for the integrated Machine Learning models.

Heart Disease

Model: Random Forest Classifier.
Dataset: UCI Cleveland Heart Disease (303 instances).
Input Vectors: 13 clinical features including CP (Chest Pain), Thal (Thalium Stress), and Oldpeak (ST Depression).
Metric: Optimized for Recall to minimize false negatives in critical cardiac events.

Kidney Disease

Model: Random Forest Classifier.
Dataset: Chronic Kidney Disease (CKD) Dataset (400 instances).
Input Vectors: 24 features derived from blood chemistry (e.g., Serum Creatinine, BUN) and urinalysis.
Performance: Achieves near-perfect classification due to distinct biomarkers in advanced CKD stages.

Skin Cancer (Dermatology)

Architecture: MobileNetV2 (Transfer Learning from ImageNet).
Preprocessing: Images are resized to 128x128px, normalized, and augmented (rotation, shear) to improve generalization.
Classes: 9 distinct conditions including Melanoma, Basal Cell Carcinoma, and Nevus.
Optimization: Uses class weighting to handle imbalanced medical data, ensuring rare malignant cases are prioritized during training.

Application Stack

  • Backend Framework: Flask (Python 3.9+)
  • Data Persistence: SQLAlchemy (SQLite) for User Management
  • ML Inference: TensorFlow 2.x (Keras) & Scikit-Learn
  • Frontend: Custom CSS3 Glassmorphism UI
